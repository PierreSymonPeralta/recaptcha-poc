# Google reCaptcha v3 implementation in NODEJS

## Intall dependencies
```javascript
npm install
```

## Add your keys
Create your __.env__ file 
```javascript
SITE_KEY='your site key'
SECRET_KEY='your secret key'
```
If don't have keys yet checkout https://developers.google.com/recaptcha/intro

## Start the app
```javascript
npm start
```
